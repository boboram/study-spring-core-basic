# 빈 스코프

## 빈 스코프란? 
- 스프링 빈이 스프링 컨테이너의 시작과 함께 생성되어서 스프링 컨테이너가 종료될 때 까지 유지되는 이유는 스프링 빈이 기본적으로 싱글톤 스코프로 생성되기 때문
  - 스코프는 빈이 존재할 수 있는 범위 
- 스프링에서 지원하는 스코프 
  - 싱글톤 : 스프링 컨테이너의 시작~종료까지 유지되는 가장 넓은 범위의 스코프
  - 프로토타입 : 프로토타입 빈의 생성과 의존관계 주입까지만 관여, 매우 짧은 범위의 스코프
  - 웹 관련 스코프 
    - request
    - session
    - application
